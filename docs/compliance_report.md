## Остаточные ограничения

Остались только ограничения, которые не снимаются локальной доработкой без внешнего изменения среды или доступа:

- неэксплицитный полный реестр всех юрлиц группы Сбер в открытом контуре;
- недоступность части ЭТП для стабильной автоматизации;
- отсутствие стабильного participant/winner-level слоя по всем источникам;
- отсутствие гарантированного live-доступа к внешнему LLM API в текущем окружении;
- отсутствие удалённого GitHub-репозитория пользователя в рамках локальной рабочей директории.
