# Documentation Index

Полный индекс актуальной документации проекта.

## Входные файлы

- `../README.md` - обзор проекта, текущая чистая статистика и быстрые команды.
- `../INDEX.md` - короткий индекс в корне.
- `../AGENTS.md` - правила работы для ИИ-агентов.
- `../Тестовое задание.txt` - исходное тестовое задание.

## Текущая рабочая документация

- `PROJECT_MAP.md` - карта директорий, кода, данных, тестов.
- `ENTITY_SCOPE.md` - политика scope юрлиц, ИНН, ОГРН, КПП, aliases и search terms.
- `SOURCES.md` - матрица источников и статус каждого источника.
- `RUNBOOK.md` - команды запуска source sprint, merge, aliases review, тестов.
- `DEDUPLICATION.md` - правила dedupe и актуальная статистика 3161.
- `CLEANUP_NOTES.md` - финальная уборка и clean batch-набор.
- `FABRIKANT_PLAN.md` - технический план реализации Fabrikant source sprint (кандидат, не реализован).

## Протоколы и старые технические заметки

- `parser_gost_v2.md` - стандарт разработки source parser V2.
- `entity_resolution_protocol.md` - старый, но всё ещё полезный протокол entity resolution.

## Архив

- `archive/` - старые аудиты, handoff notes и исторические документы. Использовать только как справку, не как текущий runbook.
